# Android App

这是一个基于 Flutter 的学习类应用示例项目，当前主要展示“一级建造师学习”首页风格的界面原型。

## 项目简介

该项目使用 Flutter 构建，当前实现了一个较完整的首页 UI 设计，包含：

- 今日练习卡片
- 学习进度统计
- 推荐学习模块
- 底部导航栏
- 课程路线标签展示

## 当前功能

当前界面中已经包含以下内容：

- 顶部标题栏和通知入口
- “今日练习”区域，展示练习入口和连续学习天数
- 已学章节、正确率等统计卡片
- 推荐模块列表，包括基础概念、工程法规、施工管理
- 底部导航栏，包含学习、题库、我的三个入口

> 说明：目前项目以界面展示和交互原型为主，按钮点击还没有真正跳转到题库或答题页面。

## 项目结构

- lib/main.dart：应用主界面入口
- pubspec.yaml：Flutter 项目依赖与配置
- test/widget_test.dart：基础测试示例

## 运行方式

### 1. 安装依赖

在项目根目录执行：

```bash
flutter pub get
```

### 2. 启动应用

如果你是在当前开发容器环境中运行，可以尝试：

```bash
flutter run -d web-server --web-port=8080
```

如果环境中已配置桌面支持，也可以使用：

```bash
flutter run -d linux
```

### 3. 访问效果

- Web 预览：http://localhost:8080

## 开发环境

- Flutter SDK：已内置于当前工作区的 flutter-sdk 目录
- 语言环境：Dart
- 目标平台：当前可在 Web Server 中预览，桌面支持需额外配置
